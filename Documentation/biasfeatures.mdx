## Adverse-Impact Matrix (AIM)

Adverse-Impact Matrix

Evaluates fairness across recruitment stages by comparing selection outcomes between demographic groups.

### Controls

* **Weighted:** Toggle to apply weighted calculations based on group size.
* **Show n:** Toggle to display sample sizes for each group.
* **Suppress &lt;N:** Input setting to hide values with insufficient sample sizes.

### Table Row Titles

* Screening
* AI Interview
* Human Review
* Offer

### Legend Description

Explains the meaning of matrix color indicators that represent fairness thresholds for comparing demographic groups.

## Parity & Drift Timeline

Parity & Drift Timeline

A line chart that visualizes fairness trends over time, tracking changes in parity and potential scoring drift.

### Controls

* **Metric Dropdown:** Selects the fairness metric to display.
* **Time Period Dropdown:** Defines whether the chart displays daily, weekly, or monthly intervals.
* **Date Range Selector:** Allows viewing trends within a specific time window.
* **Reference Lines:** Indicates target fairness levels and watch thresholds.

## Interviews (Audit/Investigation) Table

Interviews Audit Table

A review table listing interviews that triggered bias checks, with expandable details for deeper investigation.

### Table Controls

* **Export Button:** Allows downloading audit data.
* **Search Bar:** Search interviews by ID, role, or flags.
* **Filter Button:** Apply advanced filtering options.
