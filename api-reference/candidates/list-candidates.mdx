## Overview

Retrieve candidates for a specific job and organization with support for pagination, search, and status filtering.

## Endpoint

```
POST /graphql
Content-Type: application/json
```

## GraphQL Query

```graphql
query getCandidates($input: GetCandidatesInput!) {
  getCandidates(input: $input) {
    candidate {
      _id
      uId
      fullName
      emailAddress
      phoneNumber
      resumeScore
      skillsFound
      status
      createdAt
    }
  }
}
```

## Request Parameters

<ParamField body="organizationId" type="string" required>
  Filter candidates by organization
</ParamField>

<ParamField body="jobId" type="string" required>
  Filter candidates by job ID
</ParamField>

<ParamField body="pageNumber" type="integer" default={1}>
  Page number (1-based indexing)
</ParamField>

<ParamField body="pageSize" type="integer" default={10}>
  Number of items per page
</ParamField>

<ParamField body="search" type="string">
  Search term for candidate name or email
</ParamField>

<ParamField body="status" type="string">
  Filter by candidate status (e.g., "Active", "Rejected", "Hired")
</ParamField>
