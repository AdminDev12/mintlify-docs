## Overview

Schedule an interview by assigning an interviewer to a candidate with a specific date and time.

## Endpoint

```
POST /graphql
Content-Type: application/json
```

## GraphQL Mutation

```graphql
mutation assignInterviewer($input: AssignInterviewerInput!) {
  assignInterviewer(input: $input) {
    message
    result {
      acknowledged
      insertedId
      _id
    }
  }
}
```

## Request Parameters

<ParamField body="candidateId" type="string" required>
  Unique identifier of the candidate
</ParamField>

<ParamField body="interviewerId" type="string" required>
  Unique identifier of the interviewer
</ParamField>

<ParamField body="dateAndTime" type="string" required>
  Interview date and time in ISO 8601 format (UTC)
</ParamField>

<ParamField body="status" type="string" required>
  Interview status (typically "Scheduled")
</ParamField>

```python Python
import requests
from datetime import datetime, timezone

url = "https://api.exterview.ai/graphql"
headers = {
    "x-api-key": "YOUR_API_KEY",
    "Authorization": "Bearer YOUR_TOKEN",
    "Content-Type": "application/json"
}

mutation = """
mutation assignInterviewer($input: AssignInterviewerInput!) {
  assignInterviewer(input: $input) {
    message
    result {
      acknowledged
      insertedId
      _id
    }
  }
}
"""

# Schedule interview for specific date/time
interview_time = datetime(2025, 10, 22, 18, 45, 0, tzinfo=timezone.utc)

variables = {
    "input": {
        "candidateId": "C100",
        "interviewerId": "I99",
        "dateAndTime": interview_time.isoformat(),
        "status": "Scheduled"
    }
}

response = requests.post(url, headers=headers, json={"query": mutation, "variables": variables})
print(response.json())
```
