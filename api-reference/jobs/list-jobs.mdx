## Overview

Retrieve jobs for a specific organization and recruiter with support for pagination, search, and draft filtering.

## Endpoint

```
POST /graphql
Content-Type: application/json
```

## GraphQL Query

```graphql
query getJobs($input: getJobsInput!) {
  getJobs(input: $input) {
    jobs {
      _id
      title
      jobDescription
      skills
      recruiterId
      recruiterName
      organizationId
    }
  }
}
```

## Request Parameters

<ParamField body="organizationId" type="string" required>
  Filter jobs by organization
</ParamField>

<ParamField body="recruiterId" type="string" required>
  Filter jobs by recruiter
</ParamField>

<ParamField body="userType" type="string" required>
  User role (Admin/Recruiter)
</ParamField>

<ParamField body="pageNum" type="integer" default={1}>
  Page number (1-based indexing)
</ParamField>

<ParamField body="perPage" type="integer" default={20}>
  Items per page (maximum: 100)
</ParamField>

<ParamField body="search" type="string">
  Search term for job title or description
</ParamField>

<ParamField body="drafts" type="boolean" default={false}>
  Include draft jobs in results
</ParamField>
